## Mac Keyboard Modifier keys
| key  | description    |
| :--- | :------------- |
| ⌘    | ⌘ +  / Cmd     |
| ⌃    | Control / Ctrl |
| ⌥    | ⌥ + s / Alt    |
| ⇧    | Shift          |
| ⇪    | Caps Lock      |
| Fn   | Function key   |
